# BioStudy - Maria Clara Mendonça

Este projeto foi organizado para ser fácil de desenvolver e fácil de publicar no GitHub Pages.

## Estrutura do Projeto

- `index.html`: Este é o arquivo principal. Ele é **auto-contido** (contém todo o código necessário). Isso permite que o site funcione instantaneamente ao ser publicado no **GitHub Pages** sem precisar de um processo de "build".
- `src/`: Esta pasta contém os arquivos "normais" de desenvolvimento (separados por componentes, serviços e tipos). Use esta pasta se quiser fazer modificações organizadas no futuro.

## Como publicar no GitHub Pages

1. Suba todos os arquivos para o seu repositório no GitHub.
2. Vá em **Settings** > **Pages**.
3. Em **Branch**, selecione `main` (ou a sua branch principal) e a pasta `/(root)`.
4. Clique em **Save**.
5. O seu site estará disponível no link fornecido pelo GitHub em poucos minutos!

## Desenvolvimento Local

Se quiser rodar o projeto localmente usando o Vite:
1. Execute `npm install`.
2. Execute `npm run dev`.
